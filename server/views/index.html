Example #2
Requesting evaluation

curl --location --request POST 'http://juezlti.dcc.fc.up.pt/eval' --header 'Accept: application/json' --header 'Content-Type: application/json' --data-raw '{ "date":"2021-11-09T17:09:41.305Z", "program":"/bookstore/book/price//text()", "learningObject":"fd286cb3-5c95-4b0e-b843-56bc058a7713" }'

The value oflearningObject is an ID of an XPATH exercise in this repository. An example of such exercises can be found here
